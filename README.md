Object Query Language
=====================

> 


Summary
-------


Installation
------------

~~~
% npm install oql
~~~


### Dependencies ###

These are installed when **oql** is installed.

~~~
underscore: 1.x.x
rx:         2.x.x
~~~


### Development Dependencies ###

Installed when you run `npm install` in the package directory.

~~~
mocha:      1.x.x
should:     3.x.x
del:        *
gulp:       3.x.x
gulp-ejs:   0.x.x
gulp-jsdoc: 0.x.x
~~~
